視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技巧，讓網站背景的移動速度比前景慢。當訪客向下捲動網站時，這會產生 3D 效果，增加景深並帶來更沉浸的瀏覽體驗。視差基於視覺錯覺；由於人眼會把靠近的物體看得比遠處的物體大， 我們會感覺遠方的物體移動得更慢。這種錯覺長期被運用在各種媒介的視差效果中，營造逼真感。最早可追溯到 迪士尼《白雪公主和七個小矮人》等傳統動畫，以及《超級瑪利歐》等電玩遊戲。隨著 CSS 和 HTML 的進步， 視差效果才逐漸演變成今天我們在網頁設計中看到的樣貌。

2007 年，網頁開發者 Glutnix 在部落格文章中分享了最早的瀏覽器視差滾動實作之一，附上了範例程式碼與 使用 JavaScript 和 CSS 2 的示範，支援當時的 Internet Explorer 6 及其他瀏覽器。不過直到 2011 年 HTML5 與 CSS3 亮相後，這項技術才在網頁設計師之間流行起來。支持者認為，這是擁抱網頁流動性的一種簡單方式， 並以視差背景作為更好吸引用戶、提升網站整體體驗的工具。然而普渡大學在 2013 年發表的研究顯示： 「……雖然視差滾動強化了使用者體驗的部分面向，但並不一定能改善整體使用者體驗」。


為什麼要使用視差圖形？

    設計師應該嘗試視差圖形的兩個主要原因：

  • 有助於講故事

    視差圖形非常適合用來講故事。好的設計就是關於溝通——當我們設計網站時，希望與訪客交流，講述一個 關於產品、服務或品牌的動人故事。視差滾動能讓我們將訪客帶入獨特的線上體驗，甚至可以打造單頁 網站，讓訪客不用切換頁面就能讀完整個關於產品或品牌的故事。

  • 能提升使用者參與度

    視差圖形可以用來提升使用者參與度。降低網站的跳出率是網頁設計中需要留意的目標之一。跳出率代表 只瀏覽一頁就按下「返回」或關閉分頁的訪客比例。過高的跳出率會影響搜尋排名，因此任何能降低該比例 的措施都值得採用。精心打造的視差頁面能提供動態體驗，讓使用者更願意停留更久。


想在網站加入視差滾動效果時，可以考慮以下建議：

  • 選擇性地使用

    視差滾動用在特定元素上會比套用在所有頁面或整站更有效。可放在頁首、標題，或只用在首頁。你希望 視差滾動能強化網站設計，而不是分散使用者注意力。

  • 圖片壓縮

    視差滾動會使用各種媒體檔案與 CSS 形狀來營造深度感。不要讓載入過慢的大型圖片檔犧牲網站效能與 使用者體驗。

  • 善用色彩

    視差滾動不僅與前景與背景的圖片有關，也與色彩相關。背景可以使用較低彩度的色調，前景則採用更鮮明 的顏色，進一步強化深度感。
